# External Sort

Por ser muito grande o arquivo .csv que estamos trabalhando, ele está disponível aqui: https://drive.google.com/drive/folders/1WzGpaS9SNHUlrDSybl4WntzU7NMdYonE

O programa reparte o CSV em ~N~ Blocos e os ordena pela descriçao usando Quick Sort. Logo apos, ele mescla os arquivos auxiliares usando um Heap.

O arquivo escolhido foi o "Call991_2.csv"

Pontos importantes:

1) Executar o arquivo "csvToBinary.exe"
2) Executar o arquivo "port.exe"

2) Ler de uma posição a outra
